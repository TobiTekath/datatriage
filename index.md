# datatriage

Datatriage is a collection of functions to automate tabular data
formatting, type validation and detection of inconsistencies for data
cleaning workflows. It supports identifying values that do not match
their expected data type, helping to quickly locate and resolve data
quality issues.

## Installation

You can install the development version of datatriage like so:

``` r
# using pak
pak::pak("TobiTekath/datatriage")

#using remotes
remotes::install_github("TobiTekath/datatriage")
```
