# datatriage

The goal of datatriage is to help users to create reproducible data
cleaning workflows. It provides functions to automate data type
validation and detect inconsistencies in data cleaning workflows.

## Installation

You can install the development version of datatriage like so:

``` r
# using pak
pak::pak("TobiTekath/datatriage")

#using remotes
remotes::install_github("TobiTekath/datatriage")
```
